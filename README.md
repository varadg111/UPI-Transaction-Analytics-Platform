# UPI Transaction Analytics Platform

## Overview

Built an end-to-end UPI Transaction Analytics Platform processing 250,000+ transactions using Python, SQL Server, and Power BI.

## Tech Stack

* Python
* Pandas
* SQL Server
* SQLAlchemy
* PyODBC
* Power BI
* GitHub

## ETL Pipeline

1. Extract Data
2. Transform Data
3. Load into SQL Server
4. Create SQL Views
5. Build Power BI Dashboards

## Dashboard Pages

### Executive Overview

* Total Transactions
* Total Transaction Value
* Fraud Count
* Success Rate

### Fraud Analytics

* Fraud by State
* Fraud by Device Type
* Fraud by Network Type

### Bank Performance

* Transaction Volume by Bank
* Transaction Value by Bank
* Market Share Analysis

### Executive Dashboard

* Interactive Filters
* Merchant Analysis
* Monthly Trends

## Key Insights

* SBI processed the highest transaction volume.
* Shopping generated the highest transaction value.
* Maharashtra recorded the highest transaction activity.
* Fraud transactions were less than 0.2% of total transactions.
